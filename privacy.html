개인정보처리방침

시행일: 2026년 6월 21일

DotDay(이하 "본 앱")는 Windows 바탕화면에서 월간 캘린더와 일정을 확인할 수 있는 데스크톱 캘린더 위젯 앱입니다. 본 방침은 본 앱이 사용자의 캘린더 연동 정보와 일정 데이터를 어떻게 접근·이용·보관하는지 설명합니다.

1. 접근·저장하는 정보

본 앱은 사용자가 직접 캘린더 연동을 진행한 경우에 한해 필요한 정보에 접근합니다.

Google Calendar 연동

Google 로그인은 Google이 직접 처리하며, 본 앱은 사용자의 Google 계정 비밀번호에 접근하지 않습니다.

본 앱은 Google Calendar 연동을 위해 다음 정보에 접근할 수 있습니다.

  • OAuth 인증 토큰
  • Google 캘린더 목록 정보
  • Google Calendar 일정 데이터
  • 일정의 제목, 날짜·시간, 설명, 위치, 색상, 반복 정보, 참석자, 주최자, 회의 링크 등 일정 표시와 관리를 위해 필요한 정보

위 데이터는 캘린더 읽기/쓰기 권한(https://www.googleapis.com/auth/calendar)을 통해 접근합니다.

CalDAV 캘린더 연동

사용자가 CalDAV 캘린더를 직접 추가한 경우, 본 앱은 일정 표시를 위해 다음 정보에 접근할 수 있습니다.

  • CalDAV 서버 주소
  • CalDAV 계정 정보
  • CalDAV 비밀번호 또는 앱 비밀번호
  • CalDAV 캘린더 목록 정보
  • CalDAV 일정 데이터 - 일정의 제목, 날짜·시간, 설명, 위치, 색상, 반복 정보, 참석자, 주최자, 회의 링크 등 일정 표시를 위해 필요한 정보

2. 정보의 이용 목적

본 앱은 위 정보를 다음 목적으로만 이용합니다.

  • 바탕화면 위젯과 앱 화면에 사용자의 일정을 표시
  • 사용자가 요청한 경우 Google Calendar 일정의 추가·수정·삭제
  • 여러 캘린더의 일정 색상, 반복 일정, 여러 날 일정, 상세 정보를 표시
  • 캘린더 연동 상태 유지 및 일정 동기화

본 앱은 위 목적 외의 용도로 데이터를 이용하지 않으며, 광고, 프로파일링, 분석 목적으로 사용하지 않습니다.

3. 정보의 보관 및 보안

  • 캘린더 연동 정보는 사용자 기기 내부에 저장됩니다.
  • 일정 캐시는 앱 실행 중 메모리에만 임시 보관되며, 디스크에 따로 저장하지 않습니다.
  • Google 토큰과 CalDAV 비밀번호는 Windows 데이터 보호 기능(DPAPI)을 사용해 암호화하여 저장합니다.
  • 본 앱은 사용자의 일정 정보를 DotDay의 별도 서버로 보내거나 따로 수집하지 않습니다.
  • Google Calendar와 CalDAV 연동 시에는 해당 캘린더 서비스와 직접 통신합니다.
  • 오류 로그에는 토큰, 비밀번호, URL 등 민감한 정보가 남지 않도록 가려서 기록합니다.
  • Google 인증은 사용자 기기의 로컬 루프백(http://localhost) 리디렉션으로 처리됩니다.

4. 정보의 제3자 제공

본 앱은 사용자의 캘린더 데이터, 연동 정보, 인증 정보를 제3자에게 제공·판매·공유하지 않습니다.

단, 사용자가 Google Calendar 또는 CalDAV 캘린더를 연동한 경우 일정 동기화를 위해 해당 캘린더 서비스와 직접 통신합니다.

5. Google API 사용 제한(Limited Use)

본 앱이 Google API를 통해 받은 정보의 이용 및 전송은 Google API Services User Data Policy의 Limited Use 요건을 준수합니다.

DotDay's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

6. 공휴일 정보

본 앱은 한국 공휴일 표시를 위해 공공데이터포털의 "한국천문연구원 특일 정보" API에서 공휴일 데이터를 조회합니다. 공휴일 정보 조회 요청에는 사용자의 개인정보, Google Calendar 일정 데이터, CalDAV 일정 데이터를 포함하지 않습니다.

네트워크 문제로 공휴일 정보를 불러오지 못하는 경우에도 앱에 포함된 내장 데이터를 사용해 공휴일을 표시할 수 있습니다.

7. 데이터 삭제 및 연동 해제

  • 앱 내에서 Google Calendar 연동을 해제하면 기기에 저장된 Google 인증 토큰이 삭제됩니다.
  • Google 계정 권한 설정에서 본 앱의 액세스 권한을 언제든 취소할 수 있습니다.
  • 앱 내에서 CalDAV 캘린더 연동을 삭제하면 기기에 저장된 CalDAV 연결 정보가 삭제됩니다.
  • 연동 해제 시 해당 캘린더의 인증/연결 정보가 삭제되며, 앱 실행 중 메모리에 임시 보관된 일정 캐시도 함께 비워집니다.
  • 앱을 제거하면 사용자 기기에 저장된 앱 데이터는 Windows 앱 데이터 관리 방식에 따라 삭제되거나 정리될 수 있습니다.
