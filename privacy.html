개인정보처리방침

시행일: 2026년 6월 14일

DotDay(이하 "본 앱")는 바탕화면에 월간 캘린더 위젯을 표시하는 Windows 데스크톱 애플리케이션입니다. 본 방침은 본 앱이 사용자의 Google 계정 및 캘린더 데이터를 어떻게 수집·이용·보관하는지 설명합니다.

1. 접근·저장하는 정보

로그인(계정 아이디·비밀번호)은 Google이 직접 처리하며, 본 앱은 비밀번호 등 로그인 정보에 접근하지 않습니다. 본 앱은 사용자가 Google 계정 연동을 직접 진행한 경우에 한해 다음 정보에 접근합니다.

  • OAuth 인증 토큰 — Google이 발급한 캘린더 접근 토큰으로, 인증 유지를 위해 사용자 기기에 로컬 저장합니다(외부로 전송하지 않음).
  • Google 캘린더 일정 데이터 — 일정의 제목, 날짜·시간, 설명, 위치, 색상
  • 캘린더 목록 정보 — 캘린더 이름·색상 등 메타데이터

위 데이터는 캘린더 읽기/쓰기 권한(https://www.googleapis.com/auth/calendar)으로 접근합니다.

2. 정보의 이용 목적

  • 바탕화면 위젯과 앱 화면에 사용자의 일정을 표시
  • 사용자가 요청한 경우 일정의 추가·수정·삭제

본 앱은 위 목적 외의 용도로 데이터를 이용하지 않으며, 광고·프로파일링·분석 목적으로 사용하지 않습니다.

3. 정보의 보관 및 보안

  • 모든 데이터(인증 토큰, 캘린더 캐시)는 사용자 기기 내부(로컬)에만 저장됩니다(%LocalAppData%).
  • 본 앱은 사용자의 Google 데이터를 개발자의 서버나 외부 저장소로 전송·수집·저장하지 않습니다.
  • Google 인증은 사용자 기기의 로컬 루프백(http://localhost) 리디렉션으로 처리됩니다.

4. 정보의 제3자 제공

본 앱은 사용자의 Google 계정·캘린더 데이터를 어떠한 제3자에게도 제공·판매·공유하지 않습니다.

5. Google API 사용 제한(Limited Use)

본 앱이 Google API를 통해 받은 정보의 이용 및 전송은 Google API Services User Data Policy(Limited Use 요건 포함)를 준수합니다.

DotDay's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

6. 공휴일 정보

본 앱은 한국 공휴일 표시를 위해 자체 API(api.dotday.co.kr)에서 공휴일 데이터를 조회합니다. 이 통신에는 조회 연도 정보만 포함되며, 사용자의 개인정보나 Google 데이터는 전송되지 않습니다.

7. 데이터 삭제 및 연동 해제

  • 앱 내에서 Google 연동을 해제하면 기기에 저장된 인증 토큰이 삭제됩니다.
  • 또는 Google 계정 권한 설정에서 본 앱의 액세스 권한을 언제든 취소할 수 있습니다.
